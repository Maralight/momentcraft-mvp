# MomentCraft MVP

Frontend-only MVP for an interactive emotional celebration platform.

## Assets
Place these files in `/public/assets` if using Vite dev server, or `/assets` after build/static hosting:

- image1.png
- image2.png
- image3.png
- image4.png
- image5.png
- video1.mp4

For this simple GitHub Pages upload, keep them inside the included `assets` folder and also copy them into `public/assets` if you run `npm run build`.

## GitHub Pages simple hosting
This project is Vite React. For easiest GitHub deployment without terminal, use Netlify/Vercel. For GitHub Pages, build locally first or use GitHub Actions.

## Simulated MVP features
- Home/Landing
- Builder UI
- Cinematic guest experience
- Guestbook UI
- Gift centre UI
- RSVP UI
- QR code
- Customer dashboard UI
- Planner dashboard UI
- Admin dashboard UI
- Pricing UI

No database, no login, no real payments yet.
